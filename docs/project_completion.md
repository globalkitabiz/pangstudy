덱 클릭 → "+ 새 카드 추가" → 앞면/뒷면 입력

"📚 학습 시작" → 카드 학습 → 난이도 선택

🌙 버튼 클릭 → 다크 모드 전환

검색창에 덱 이름 입력 → 실시간 필터링

### 7. 덱 공유
"📤 이 덱 공유하기" → 링크 복사 → 친구에게 전달
